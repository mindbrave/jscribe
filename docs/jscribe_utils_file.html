function discover_files(input_paths, input_regex, ignore_paths_regex=[], ignore_regex=None)

  • input_paths {list}

    List of paths to directories that contains your files you want to be discovered.

  • input_regex {str}

    regex that matches file names that should be discovered

  • ignore_paths_regex {list} Default: []

    list of regexes that match paths to directories that you want to exclude from your input_paths

  • ignore_regex {str} Default: None

    regex that matches source file names that must not be discovered

returns {list}

List of discovered filepaths

Return list of filepaths that align with given entry parameters (input paths, file regex, ignores).
